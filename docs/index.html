Four commands. One operating system.

Each is a Claude Code skill that activates automatically when you ask.

/ads audit

Audit

Grades any account against 250 rules and returns a fix list ranked by dollar impact — not by what's easy to spot.

/ads plan

Plan

Builds a media plan, budget split, honest forecast range, and a 90-day testing roadmap from a goal and a budget.

/ads create

Create

Writes platform-spec ad copy — RSAs, Meta, LinkedIn, TikTok — starting from strategic angles, with a testing matrix.

/ads compete

Compete

Competitive teardown → messaging map of crowded vs. open space → the wedge angle you can credibly own.

100% free & open source

The whole audit engine — every one of the 250 rules — is free, MIT-licensed, and yours to use, fork, and improve. No paywall, no trial, no account. Run it on your own campaigns or your clients', as many times as you like.

  1. Install the open-source plugin in Claude Code — free.
  2. Audit any account against all 250 rules.
  3. Fix what's losing you money, in priority order.

AdSignal audits and advises only — it never edits, pauses, or spends inside a live ad account. You stay in control.
